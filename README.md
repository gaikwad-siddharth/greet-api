# Greet-API

Greet-API is a Spring Boot REST application that exposes greeting APIs and is integrated with **Spring Boot Actuator** and **Spring Boot Admin Client** for application monitoring and management.

---

## Features

- RESTful Greeting API
- Spring Boot Actuator
- Spring Boot Admin Client Integration
- Health Monitoring
- Metrics Collection
- Environment Information
- Application Monitoring Dashboard

---

## Technologies Used

- Java 17+ (or your project's Java version)
- Spring Boot
- Spring Boot Actuator
- Spring Boot Admin Client
- Maven
